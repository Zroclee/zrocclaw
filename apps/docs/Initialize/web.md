# UI 组件库应用

## UI 应用初始化
1. 基于 Vite 初始化 UI 库
```bash
cd apps
pnpm create vite
│
◇  Project name:
│  web
│
◇  Select a framework:
│  Vue
│
◇  Select a variant:
│  TypeScript
│
◇  Install with pnpm and start now?
│  No
│
◇  Scaffolding project in xxx/apps/web...
│
└  Done. Now run:

  cd web
  pnpm install
  pnpm dev
```
